Chaya 

Chaya is a beginner-friendly programming language built using Python.

It is designed to make programming easier to read and learn through simple syntax and tea-inspired commands.

Current Version: v1.3

---

Features

Variables

Store values using "keep"

Example:

keep age as 18

say age

---

Output

Display text using "say"

Example:

say "Hello World"

---

Input

Take user input using "ask"

Example:

ask name

say name

---

Conditions

Supports:

- if
- else
- done

Example:

keep age as 20

if age >= 18
say "Adult"
else
say "Child"
done

---

Loops

Repeat blocks using "repeat"

Example:

repeat 3
say "Hello"
done

---

Functions

Create reusable code using:

- brew
- serve

Example:

brew greet
say "Welcome"
done

serve greet

---

String Concatenation

Example:

keep name as Pragya

say "Hello " + name

Output:

Hello Pragya

---

Lists

Create:

keep fruits as [Apple, Mango]

Access:

say fruits[0]

Add:

add Banana to fruits

Remove:

remove Mango from fruits

Size:

size fruits

---

Installation

Clone repository:

git clone YOUR_REPOSITORY_URL

Open project:

cd Chaya

Run:

python main.py
