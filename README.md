# Chaya
A beginner - friendly programming language built in Python which is easier than Python.

## Features

- Variables (keep)
- Input (ask)
- Output (say)
- Math Operations
- Conditions (if else done)

## Example

keep x as 10
keep y as 4

say x + y

Output:

14
